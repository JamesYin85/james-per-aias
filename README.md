# james-per-aias
## 安装crewAI和依赖项
```
# 创建crew虚拟环境
python -m venv crew
# 激活虚拟环境
source crew/bin/activate
# 安装crewAI
pip install crewai
# 安装依赖库
vim requirements.txt
# requirements.txt文件中填入依赖项
unstructured
langchain
Jinja2>=3.1.2
click>=7.0
duckduckgo-search
# 安装依赖库
pip install -r requirements.txt
```
